# Document Request Efficiency

This is a web-based project that demonstrates an **efficient document request interface**, created using **HTML, CSS, and JavaScript**.  
Originally developed on **CodePen**, now uploaded to GitHub and live via **GitHub Pages**.

## Features

- Clean and simple UI for submitting document requests
- Responsive design that works on desktops and mobile devices
- Full front-end implementation with HTML, CSS, and JS

## How to Open

There are two ways to view this project:

### **Option 1: View online (recommended)**
1. Click the **Live Demo** link above.
2. Your browser will open the project exactly like CodePen.

### **Option 2: Open locally**
1. Clone or download this repository.



2. Navigate to the project folder on your computer.  
3. Open `index.html` in your browser.  
- On desktop: double-click the file or right-click → Open with browser  
- On mobile: use a file browser app that can open HTML files
